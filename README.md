# Custom Start/Home Page (Multi LIVE Search) with Live Animated Weather and News Ticker

A minimalist, self-hosted start page featuring live weather updates, a custom news ticker, and multi-site live search functionality. This project is fully customizable and mobile-ready, designed to be lightweight and user-friendly.

## Demo

Check out the live demo [here](https://seanvree.github.io/homepage/).

## Features

- **Multi-Live Search**: Search Google, YouTube, Wikipedia, IMDB, and more from a single input box.
- **Live Weather Updates**: Weather data is fetched using your geolocation via the OpenWeatherMap API.
- **Customizable News Ticker**: Displays a live RSS news feed from `feed.mikle.com`.
- **Mobile-Ready**: Fully responsive design that works well on mobile and desktop devices.
- **Bookmarks**: Customizable bookmarks for quick access to your favorite sites.
- **Date and Time Display (DTG)**: Shows live date and time with click-to-convert between 12/24-hour formats.
- **Dynamic Background**: Background image auto-switches between day and night based on local browser time.
- **Calendar (Desktop Only)**: A monthly calendar is displayed on desktop browsers with large screens.
- **Page Hit Counter**: Optional PHP-based page hit counter.
- **Google Analytics Integration**: Track page views with Google Analytics.
